## Deployment

### Surge.sh

___Prefered way___

* `npm i -g surge`
* `npm run surge`


### Heroku

Heroku works out of the box, just use "deploy to heroku" button


## Keep it up to date

Track this repo

```
git remote add upstream https://github.com/werein/elm.git
```

Rebase on the latest version

```
git fetch upstream
git rebase upstream/master
```
